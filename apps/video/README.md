# Astro Stack video

Remotion source for Astro Stack product videos. `AstroStackCliDemo` is an
18-second, 1280×720 terminal demo of a marketing project being created with
the CLI.

## Commands

Start the Remotion studio:

```sh
pnpm --filter @astro-stack/video dev
```

Render the CLI demo:

```sh
pnpm --filter @astro-stack/video render:cli-demo
```

The rendered MP4 is written to `out/astro-stack-cli-demo.mp4`.
